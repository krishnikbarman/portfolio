# Krishnik Barman — Portfolio

Personal portfolio website built with Next.js, Tailwind CSS, and TypeScript.

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```
